# DELI-cious

DELI-cious Order System

This code creates a simple order management system for a deli called DELI-cious. This code allows users to create orders by adding items like sandwiches, drinks, and chips, view the total cost, and save your order as a receipt.

Features:

Create an Order:
Users can add sandwiches, drinks, and chips to their order.

Item Choices:

- Sandwiches: Choose bread type and size (4", 8", 12"), with different prices for each size.

- Drinks: Choose drink size (small, medium, or large), each with its own price.

- Chips: Select from different chip types (e.g., Original, BBQ), with a fixed price.


Checkout:
- The system calculates the total cost of the order and generates a receipt.


File Saving:
- Receipts are saved as text files in a folder called receipts.



Simple Interface:

- Users interact with the system using a menu with options to add items or complete the order.
